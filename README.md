# Xavier + Runic X

Xavier + Runic X is a gradient-free structural search engine with declarative control.

## Core ideas
- gradient-free search
- structural traversal instead of classical descent
- declarative control through Runic X
- benchmark performance on Sphere, Rosenbrock, Rastrigin, and Ackley
